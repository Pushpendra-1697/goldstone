# goldstone

## Installation

Clone the Repository from Github. Then do the following steps:

```bash
    npm install

For Frontend
    npm run start
For Backend    
    npm run server
```
# Tech Stack:
React.js, Redux, node.js, express.js, mongoDB

# Frontend Live link:
https://goldstone-xi.vercel.app
# Backend server link:
